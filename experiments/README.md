# experiments/ — Research Notebook / Experiment Registry

> [!] Research Only. No Real Orders. Production Trading: BLOCKED.

This directory contains **runtime experiment outputs** generated by the TW Quant Cockpit research platform. It is **not committed to git** (see `.gitignore`). `experiments/README.md` and `experiments/.gitkeep` may be committed; all `EXP-*` subdirectories and `registry.json` are excluded.

---

## experiment_id Format

```
EXP-YYYYMMDD-HHMMSS-shortuuid
```

Example: `EXP-20260601-143022-a1b2c3d4`

---

## Directory Layout

```
experiments/
├── registry.json                  ← index of all experiments (excluded from git)
├── README.md                      ← this file
├── .gitkeep
└── EXP-20260601-143022-a1b2c3d4/
    ├── metadata.json              ← experiment metadata
    ├── notes.md                   ← appended notes
    ├── notebook.md                ← generated research notebook
    ├── snapshots/                 ← point-in-time JSON snapshots
    │   ├── config.json
    │   ├── universe.json
    │   ├── data_quality.json
    │   ├── rule_governance.json
    │   └── backtest.json
    └── reports/                   ← report path references
```

## CLI Quick Reference

```bash
# Create a new experiment
python main.py experiment-create --name "my research" --type daily_research

# Register the latest run against the most recent experiment
python main.py experiment-register-latest --mode real

# List all experiments
python main.py experiment-list

# Show a specific experiment
python main.py experiment-show --id EXP-xxxx

# Build all snapshots for an experiment
python main.py experiment-snapshot --id EXP-xxxx

# Generate the notebook
python main.py experiment-notebook --id EXP-xxxx

# Compare two experiments
python main.py experiment-compare --ids EXP-aaa,EXP-bbb

# Generate registry report
python main.py experiment-report
```
