
REVIEW the crash reversal summary. Identify stocks with strong post-crash stabilization.

```

```

REVIEW scores for relative strength and stabilization conditions.

## Step 3 — Review Stabilization Checklist

```
python main.py crash-reversal --mode real
```

REVIEW the 8-condition stabilization checklist:
1. Volume drying up
2. Price stabilization
3. Relative strength vs market
4. EPS-backed recovery
5. Moving average discipline
6. Industry sector health
7. Fundamental check
8. Risk guard conditions

```
python main.py crash-reversal-watchlist --mode real
```

KEEP_OBSERVING stocks on the watchlist. No trading action.


READ_REPORT the generated crash reversal report. Archive for review.


REVIEW the 6-stage crash reversal evidence chain in the evidence graph.
