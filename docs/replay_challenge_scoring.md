# Replay Challenge Scoring v1.2.7

**[!] Profit != high score. Loss != low score. Process weight >= Outcome weight.**
**[!] Research Only. No Real Orders. Not Investment Advice.**

## Default Weights

| Component | Weight |
|-----------|--------|
| Process Quality | 35% |
| Discipline | 15% |
| Risk Planning | 15% |
| Information Usage | 10% |
| Strategy Awareness | 10% |
| MTF Awareness | 10% |

## Outcome

- Default: NOT included in active challenge
- Post-review: optional addition, max 20%
- Process weight always >= Outcome weight

## Classifications

- EXCELLENT_PROCESS (process >= 80)
- GOOD_PROCESS (process >= 65)
- MIXED_PROCESS (process >= 50)
- WEAK_PROCESS (process < 50)
- PROCESS_ONLY (no outcome)
- GOOD_PROCESS_BAD_OUTCOME
- BAD_PROCESS_GOOD_OUTCOME
- INSUFFICIENT
- BLOCKED
